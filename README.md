## 如何使用

### 方式一：直接打开
双击 `index.html` 即可在浏览器中打开。

### 方式二：VS Code Live Server（推荐）
1. 在 VS Code 中安装 Live Server 插件
2. 右键 `index.html` → 选择 "Open with Live Server"
3. 浏览器会自动打开

## 技术栈

- 原生 HTML5 + CSS3 + JavaScript
- 纯前端，数据存储在浏览器的 localStorage 中
- 响应式设计，手机电脑都能用
- 无需服务器，无需数据库

## 自定义

如需修改配置，打开 `js/app.js` 文件，找到 `CONFIG` 对象：

```javascript
const CONFIG = {
    PASSWORD: '0616',           // 登录密码
    GIRL_NAME: '希宝',          // 女朋友的称呼
    BOY_NAME: '小李',           // 你的称呼
    START_DATE: '2021-06-26',   // 在一起的日子
    GIRL_BIRTHDAY: { month: 6, day: 16 },  // 她的生日
};
```
